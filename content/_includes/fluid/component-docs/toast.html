Toast

Improve this Doc Toast is a subtle notification that appears on top of an app's content. Typically, Toasts are displayed for a short duration of time then automatically dismiss. _For more information, Check out the [API docs](../api/components/toast/ToastController)._

Basic Usage

Demo Demo Source

```typescript import { ToastController } from 'ionic-angular'; export class MyPage { constructor(public toastCtrl: ToastController) { } presentToast() { const toast = this.toastCtrl.create({ message: 'User was added successfully', duration: 3000 }); toast.present(); } } ```
