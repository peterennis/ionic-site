<ol class="installation">
  <li>Install the Cordova and Ionic Native plugins:<br>
    <pre><code class="nohighlight">$ ionic cordova plugin add cordova-sms-plugin
$ npm install --save @ionic-native/sms
</code></pre>
  </li>
