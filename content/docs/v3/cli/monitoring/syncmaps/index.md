# `$ ionic monitoring syncmaps`

Sync Source Maps to Ionic Pro Error Monitoring service
## Synopsis

```bash
$ ionic monitoring syncmaps [<snapshot_id>]
```
  
## Details


Input | Description
----- | ----------
`snapshot_id` | An Ionic Pro snapshot id to associate these sourcemaps with.
