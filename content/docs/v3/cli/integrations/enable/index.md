# `$ ionic integrations enable`

Add various integrations to your app
## Synopsis

```bash
$ ionic integrations enable [<id>]
```
  
## Details


Input | Description
----- | ----------
`id` | The integration to enable (`cordova`, `gulp`)


Option | Description
------ | ----------
`--quiet` | Do not log file operations
